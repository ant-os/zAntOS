# AntOS Kernel Sourcetree

This directory contains the zig source code for the zAntOS Kernel,
it is organized into mainly the following subsystems:

- `mm` Memory Manager
- `io` Input/Output, Devices and Drivers types
- `antk` AntOS Kernel API
- `hal` Hardware Abstraction Layer
- `ke` Kernel Executive
- `ob` Object Manager
- `sched` Process/Thread Scheduler
